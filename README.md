# MPU-6050-RGB-Motion-Visualizer
🎨 MPU-6050 RGB Motion VisualizerA creative way to visualize sensor data! This project uses an Arduino Uno and an MPU-6050 to map real-time gyroscope and accelerometer tilt data directly to an RGB LED, creating an interactive "light show" that reacts to your movement.  🛠 HardwareArduino Uno R3  MPU-6050 (Gyroscope/Accelerometer module)  RGB LED (Common Cathode recommended)  Breadboard & Jumper Wires  ⚡ WiringComponent PinArduino PinMPU-6050 VCC5VMPU-6050 GNDGNDMPU-6050 SDAA4MPU-6050 SCLA5RGB LED (Red)D9RGB LED (Green)D10RGB LED (Blue)D11🚀 Getting StartedInstall the Adafruit MPU6050 and Adafruit Unified Sensor libraries via the Arduino Library Manager.  Connect your components as shown in the wiring table above.  Upload the MPU_RGB_Visualizer.ino file provided in this repository.Tilt the sensor to see the LED colors shift in real-time.  💡 NotesIf the LED colors seem "inverted" compared to your tilting direction, you can simply swap the pin assignments for the Red, Green, and Blue pins in the code to match your preference.👥 CreditsBuilt by CodetoCircut
